# 🎨 每日文化播客 Daily Culture Podcast

一个自动生成文化类播客的系统，每天推送艺术、历史、文学、音乐等深度内容。

## ✨ 特性

- 🎭 **每日自动更新**：每天生成新的文化主题播客
- 🎵 **高质量音频**：使用微软Edge TTS生成自然语音
- 📱 **视频号支持**：自动生成适合微信视频号的短视频
- 📡 **RSS订阅**：生成播客RSS源，可在各大平台订阅
- ⚙️ **完全自动化**：通过GitHub Actions每日自动运行
- 📚 **丰富内容库**：包含艺术、历史、文学、音乐等多个主题

## 📦 快速开始

### 安装

git clone https://github.com/你的用户名/daily-culture-podcast.git
